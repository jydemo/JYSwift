# JCYouku

### 这个是我在学习Swift过程中利用业余时间边学边做的山寨优酷项目，希望可以帮助到想学习Swift的人。
### 原版的首页板块数量较多，我只做了2个板块，也就是可重用的cell，要增加板块只要套我写好的模板增加新的cell就可以了。
### 所有数据都是通过接口获得的，以达到更加真实的效果。
